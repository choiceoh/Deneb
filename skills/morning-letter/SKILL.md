# 모닝레터

매일 아침 발송하는 일일 브리핑 레터.

## 실행 조건

- 크론잡 `morning-letter`가 매일 08:00 KST에 자동 트리거
- 사용자가 "모닝레터", "아침 브리핑" 등을 요청할 때 수동 실행

## 실행 방법

`morning_letter` 도구를 호출한다. 파라미터 없이 호출하면 6개 섹션(날씨, 환율, 구리시세, 일정, 메일, SMP 전력가격)을 병렬 수집하여 포맷된 한국어 텍스트를 반환한다.

```json
{"tool": "morning_letter"}
```

## 결과 전달

1. `morning_letter` 도구 결과를 그대로 사용자에게 전송한다
2. 결과 텍스트를 수정하거나 재구성하지 않는다
3. 추가 인사말이나 설명을 덧붙이지 않는다
4. 중간 상태 메시지("수집 중", "처리 중" 등)를 출력하지 않는다
5. 도구 호출이 실패한 경우에만 "모닝레터 생성에 실패했습니다"라고 알린다
