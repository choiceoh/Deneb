# 모닝레터

매일 아침 발송하는 일일 브리핑 레터. 6개 섹션의 데이터를 수집하여 텔레그램 메시지로 포맷팅한다.

## 실행 조건

- 크론잡 `morning-letter`가 매일 08:00 KST에 자동 트리거
- 사용자가 "모닝레터", "아침 브리핑" 등을 요청할 때 수동 실행

## 데이터 수집 (병렬 실행 권장)

각 섹션은 독립적으로 수집한다. 한 섹션이 실패해도 나머지는 정상 진행.

### 1. 날씨 (광주광역시)

`exec` 도구로 wttr.in JSON API 호출:

```bash
curl -s "wttr.in/Gwangju,South+Korea?format=j1"
```

JSON 응답에서 추출:
- `current_condition[0]`: 현재 기온(`temp_C`), 체감온도(`FeelsLikeC`), 날씨상태(`lang_ko[0].value`), 습도(`humidity`)
- `weather[0]`: 오늘 최저(`mintempC`), 최고(`maxtempC`)
- `weather[0].hourly`: 강수확률(`chanceofrain`) 높은 시간대 확인

### 2. 환율

`web` 도구로 검색:

```json
{"query": "USD KRW EUR KRW 환율 오늘 실시간"}
```

추출 항목:
- USD/KRW 매매기준율 + 전일 대비 변동
- EUR/KRW 매매기준율 + 전일 대비 변동

### 3. 구리시세

`web` 도구로 검색:

```json
{"query": "LME copper price today USD per ton 구리 시세"}
```

추출 항목:
- LME 구리 가격 (USD/ton) + 전일 대비 변동률

### 4. 오늘 일정

`exec` 도구로 gcalcli 호출:

```bash
gcalcli agenda "today" "tomorrow" --nostarted --details length
```

- gcalcli 미설치 시: "일정 조회 불가 (gcalcli 미설치)" 표시
- 일정 없으면: "오늘 일정 없음" 표시
- 일정 있으면: 시간 + 제목 목록 (최대 8건)

### 5. 전일 메일 주요사항

`gmail` 도구로 어제 수신 메일 조회:

```json
{"action": "search", "query": "newer_than:1d", "max": 15}
```

- 검색 결과에서 중요도/발신자 기준 상위 5건 선별
- 각 메일: 발신자 + 제목 1줄 요약
- 메일 없으면: "어제 수신 메일 없음" 표시

### 6. 한국 SMP 전력가격

`web` 도구로 검색:

```json
{"query": "한국 SMP 전력시장 가격 전력거래소 KPX 오늘"}
```

- 육지 SMP 가격 (원/kWh) + 전일 대비 변동
- 조회 실패 시: "SMP 가격 조회 불가" 표시

## 출력 포맷

텔레그램 메시지로 발송. 총 4000자 이내로 작성.

### 템플릿

```
🌅 모닝레터 — {YYYY년 M월 D일} {요일}

☁️ 날씨 (광주광역시)
현재 {기온}°C (체감 {체감}°C), {날씨상태}
최저 {최저}° / 최고 {최고}°, 습도 {습도}%
{강수확률 30% 이상이면: "🌧 강수확률 {확률}% ({시간대})"}

💱 환율
USD/KRW: {매매기준율} ({변동 +/-})
EUR/KRW: {매매기준율} ({변동 +/-})

🔶 구리시세
LME: ${가격}/ton ({변동률})

📅 오늘 일정
{• 시간 — 제목} (각 줄)
{또는 "일정 없음"}

📧 전일 메일 주요사항
{• 발신자 — 제목 요약} (각 줄, 최대 5건)
{또는 "수신 메일 없음"}

⚡ SMP 전력가격
육지: {가격}원/kWh ({변동})
{또는 "조회 불가"}

좋은 하루 되세요!
```

## 에러 처리 규칙

1. 섹션별 독립 수집 — 한 섹션 실패가 전체를 중단시키지 않음
2. 실패한 섹션은 "조회 실패" 또는 "조회 불가"로 표시
3. 모든 섹션이 실패해도 날짜/인사말이 포함된 기본 레터는 발송
4. 네트워크 타임아웃: 각 도구 호출에 15초 제한 권장

## 포맷 규칙

- 한국어로 작성
- 숫자에 천단위 콤마 사용 (예: 1,350.50)
- 변동률은 +/- 부호 포함 (예: +1.2%, -15.30)
- 환율/시세의 전일 대비: 상승은 "▲", 하락은 "▼", 보합은 "—"
- 간결하게: 각 섹션 3줄 이내
- 총 길이 4000자 초과 시 메일 요약을 3건으로 축소
