The gateway binary embeds this placeholder so the //go:embed compile step does not fail when the Mini App frontend has never been built. The real Mini App is served from miniapp_dist/index.html (and assets/), both gitignored and regenerated by make embed-frontend.

To replace this page with the real Mini App, run make build-frontend && make embed-frontend at the repo root, then rebuild the gateway. Production deploys do this via scripts/deploy/deploy.sh.
