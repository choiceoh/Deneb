## Major Hidden Features

### KAIROS — Proactive Agent Mode

**Flags**: `PROACTIVE`, `KAIROS` (compile-time)

A persistent, always-running Claude assistant that acts without user prompts.

#### Architecture
- **Append-only daily logs**: Track observations and decisions
- **Tick prompts**: Periodic evaluation — "should I act now?"
- **15-second blocking budget**: Defers actions exceeding user impact threshold
- **Brief Mode output**: Ultra-concise responses from persistent assistant

#### Exclusive Tools
| Tool | Purpose |
|------|---------|
| SendUserFile | Push files to user |
| PushNotification | Mobile/desktop notifications |
| SubscribePR | Watch GitHub PR for changes |

#### Behavior
- Watches file changes, build outputs, test results
- Proactively acts on patterns it notices
- Can subscribe to GitHub PR events and auto-respond
- Operates with minimal context window to stay fast

### ULTRAPLAN — Remote Deep Planning

**Flag**: `tengu_ultraplan_model`

Offloads complex planning to a remote container:

#### Flow
1. Detect complex task requiring deep thought
2. Spin up Cloud Container Runtime (CCR) session
3. Run Opus 4.6 with **30-minute thinking budget**
4. Terminal shows polling state (3-second intervals)
5. Browser-based UI for real-time observation
6. User approves or rejects plan
7. Sentinel `__ULTRAPLAN_TELEPORT_LOCAL__` teleports result back to terminal

#### Properties
- Remote execution in container (not local)
- Much longer thinking budget than normal (30 min vs seconds)
- Interactive browser UI for plan review
- Explicit user approval gate before execution

### Buddy System — Tamagotchi Companion

**Flag**: `BUDDY` (compile-time)

A deterministic gacha system with reproducible outcomes.

#### RNG System
- **Mulberry32 PRNG** seeded from `userId hash + salt 'friend-2026-401'`
- Deterministic: same user always gets same buddy
- Shiny chance: 1% (independent of rarity)

#### Species (18 total, 5 rarities)

| Rarity | Chance | Species |
|--------|--------|---------|
| Common (60%) | 60% | Pebblecrab, Dustbunny, Mossfrog, Twigling, Dewdrop, Puddlefish |
| Uncommon (25%) | 25% | Cloudferret, Gustowl, Bramblebear, Thornfox |
| Rare (10%) | 10% | Crystaldrake, Deepstag, Lavapup |
| Epic (4%) | 4% | Stormwyrm, Voidcat, Aetherling |
| Legendary (1%) | 1% | Cosmoshale, Nebulynx |

Shiny Legendary: 0.01% chance.

#### Properties
- 5 procedural stats: DEBUGGING, PATIENCE, CHAOS, WISDOM, SNARK (0-100)
- 6 eye styles, 8 hat options (some rarity-gated)
- Soul description generated by Claude on first hatch
- 5-line ASCII art sprites with animation frames (idle, reaction)
- Teaser window: April 1-7, 2026; Full launch: May 2026

## Remote Configuration

### Polling
- Claude Code polls Anthropic servers every hour
- Can push configuration changes to running instances
- Activate/deactivate features via GrowthBook
- No user-initiated update required
- No explicit consent for flag changes

### Kill Switches
Each major feature has a corresponding kill switch:
- `tengu_penguins_off` → disables fast mode
- Various per-feature disable flags
- Server-side control over client behavior
