# `deneb setup`

Initialize `~/.deneb/deneb.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- CLI onboarding: [Onboarding (CLI)](/start/wizard)

## Examples

```bash
deneb setup
deneb setup --workspace ~/.deneb/workspace
```

To run onboarding via setup:

```bash
deneb setup --wizard
```
